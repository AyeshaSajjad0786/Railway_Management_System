# Train Management System

## Description
A Java-based Train Management System built using Object-Oriented Programming (OOP).  
It manages users, trains, and data storage using file handling and core OOP concepts.

## Features
- User registration and management
- Train scheduling system
- Data storage using serialization
- Simple and modular structure

## OOP Concepts Used
- Encapsulation → Private data with getters/setters
- Inheritance → Interfaces for common behavior
- Polymorphism → Method overriding in handlers
- Abstraction → Hidden file handling logic

## Technologies
- Java
- NetBeans
- File Handling (users.dat)
- Swing (GUI)
- java.time API

## Classes
- User
- Train
- UserDataHandler

## Outcome
A working OOP-based system that manages train and user data in a structured and efficient way.
